# 技能数独
技能数独就是在传统数独里加入技能好好玩！
一个由AI史山代码和一点点幼稚的想法构成的可联机对战小游戏~

## 功能特点

- 三种难度等级：简单、普通、困难
- 笔记功能：可以在格子上记录可能的数字
- 生命值系统：玩家有5条生命，错误答案会扣除生命值
- 正确答案提示：填写正确的数字会变成绿色
- 联机对战：支持两人同时游戏，比拼完成速度
- 实时进度显示：可以看到对手的游戏进度

## 如何运行

- 将所有文件放在同一个目录中
- 安装Node.js和npm
- 在项目目录中运行 npm install 安装依赖
- 运行 npm start 启动服务器
- 在浏览器中本地访问 http://localhost:3000
- 或远程访问(记得在防火墙等放通端口)http://ip:3000
